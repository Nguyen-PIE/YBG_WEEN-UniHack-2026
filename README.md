# YBG_WEEN-UniHack-2026
UniHack 48 Hour Hackathon. 5 Builders. 1 Dream. Fighting food security and cost of living crisis on the local level. 

https://budget-bunny-uni-hack-2026.vercel.app/

## Vercel server-side (Elasticsearch)
- Set `ELASTIC_URL` and `ELASTIC_API_KEY` in your Vercel project environment variables.
- For local development with serverless functions, run `vercel dev` from `frontend`.
- The frontend calls `/api/generate-recipe` and `/api/manual-list` on the same origin.
